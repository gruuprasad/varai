# Varai Map — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace `varai audit` with `varai map <repo>` — a clean inventory of what a repo contains (routes, models, stores, packages, env vars), organized by category, no intent file required.

**Architecture:** The scanner becomes a registry of per-stack extractors (nextjs, fastapi, sqlalchemy, react-vite, python-common) behind a stack-detection step. A new inventory renderer replaces the intent-coverage reporter. The extractor interface is `(repoPath, files) → facts[]` so the Phase 2 SCIP backbone can slot in later without touching consumers.

**Extraction engine:** The new Python/React extractors parse with **tree-sitter** (`web-tree-sitter` + prebuilt `tree-sitter-wasms` grammars — WASM, no native compilation), not regex over raw text. The pattern throughout: a stable tree-sitter query isolates the real node (`(decorator)`, `(class_definition)`, `<Route>`), then a small read pulls the identifying parts out of that *parser-validated* node — so comments, strings, and multiline formatting can't produce false facts. Facts from a parse tree are tagged `layer: "ast"`. The existing Next.js extractor stays path/manifest-based (`layer: "heuristic"`) — its routes come from reliable file-path conventions, not lying heuristics, and its golden tests must stay green.

**Tech Stack:** Node.js ESM, `web-tree-sitter` + `tree-sitter-wasms`, `node:test` + `node:assert/strict` for tests

## Fact shape (used across all tasks)

Every fact returned by an extractor has this shape:

```js
{
  kind: string,         // "api_route" | "db_model" | "page" | "state_store" | "package" | "env_var" | "database_migration" | "webhook_route" | "component" | "code_hint"
  name: string,         // human-readable name
  detail?: string,      // optional extra info (e.g. package version)
  tags?: string[],      // optional semantic tags
  evidence: [{ file: string, line?: number }],
  layer: "ast" | "heuristic"   // "ast" = tree-sitter verified; "heuristic" = path/manifest/text
}
```

**Layer tagging rule:** facts produced by a tree-sitter parse (FastAPI routes, SQLAlchemy models, React routes/stores) are `"ast"`. Facts read from file-path conventions, manifests, or plain text (Next.js routes, Prisma schema, package lists, env-var names) are `"heuristic"`. `"semantic"` is reserved for the Phase 2 SCIP graph and is not emitted here.

```js
import path from "node:path";
import { createRequire } from "node:module";
import Parser from "web-tree-sitter";

const require = createRequire(import.meta.url);
const WASM_DIR = path.join(path.dirname(require.resolve("tree-sitter-wasms/package.json")), "out");

let initPromise;
const langCache = new Map();

async function ensureInit() {
  if (!initPromise) initPromise = Parser.init();
  await initPromise;
}

async function loadLanguage(lang) {
  if (!langCache.has(lang)) {
    await ensureInit();
    const Lang = await Parser.Language.load(path.join(WASM_DIR, `tree-sitter-${lang}.wasm`));
    langCache.set(lang, Lang);
  }
  return langCache.get(lang);
}

// Parse `code` in `lang`, run `queryString`, return captures [{ name, node }].
// node.text = source text of the node; node.startPosition.row = 0-based line.
export async function queryCaptures(lang, code, queryString) {
  const Lang = await loadLanguage(lang);
  const parser = new Parser();
  parser.setLanguage(Lang);
  const tree = parser.parse(code);
  const query = Lang.query(queryString);
  return query.captures(tree.rootNode);
}

// For the spike only: print the parse tree of a snippet.
export async function debugTree(lang, code) {
  const Lang = await loadLanguage(lang);
  const parser = new Parser();
  parser.setLanguage(Lang);
  return parser.parse(code).rootNode.toString();
}
```

```js
import path from "node:path";
import { readFile, stat } from "node:fs/promises";
import { dedupeFacts } from "../utils.js";

const TEXT_FILE_EXTENSIONS = new Set([
  ".cjs", ".css", ".env", ".js", ".jsx", ".json", ".md", ".mjs",
  ".prisma", ".sql", ".ts", ".tsx", ".txt", ".yaml", ".yml"
]);

export async function extract(repoPath, files) {
  const facts = [];
  facts.push(...await packageFacts(repoPath));
  facts.push(...await fileShapeFacts(repoPath, files));
  facts.push(...await prismaFacts(repoPath, files));
  facts.push(...await envFacts(repoPath, files));
  return facts;
}

async function packageFacts(repoPath) {
  const packagePath = path.join(repoPath, "package.json");
  try {
    const contents = await readFile(packagePath, "utf8");
    const parsed = JSON.parse(contents);
    const dependencies = { ...(parsed.dependencies ?? {}), ...(parsed.devDependencies ?? {}) };
    return Object.keys(dependencies).map((name) => ({
      kind: "package",
      name,
      detail: dependencies[name],
      tags: packageTags(name),
      evidence: [{ file: "package.json" }],
      layer: "heuristic"
    }));
  } catch {
    return [];
  }
}

async function fileShapeFacts(repoPath, files) {
  const facts = [];
  for (const file of files) {
    if (isNextPage(file)) {
      facts.push({ kind: "page", name: routeNameFor(file), evidence: [{ file }], layer: "heuristic" });
    }
    if (isNextApiRoute(file)) {
      const routeName = routeNameFor(file);
      facts.push({ kind: "api_route", name: routeName, evidence: [{ file }], layer: "heuristic" });
      if (/webhook/i.test(routeName) || /webhook/i.test(file)) {
        facts.push({ kind: "webhook_route", name: routeName, evidence: [{ file }], layer: "heuristic" });
      }
    }
    if (file.startsWith("components/") || file.includes("/components/")) {
      facts.push({ kind: "component", name: path.basename(file, path.extname(file)), evidence: [{ file }], layer: "heuristic" });
    }
    if (file.startsWith("supabase/migrations/")) {
      facts.push({ kind: "database_migration", name: path.basename(file), evidence: [{ file }], layer: "heuristic" });
    }
  }
  await maybeAddTextHints(repoPath, files, facts);
  return facts;
}

function isNextPage(file) {
  return /^app\/(?:.+\/)?page\.(jsx|tsx|js|ts)$/.test(file) || /^pages\/.+\.(jsx|tsx|js|ts)$/.test(file);
}

function isNextApiRoute(file) {
  return /^app\/api\/.+\/route\.(js|ts)$/.test(file) || /^pages\/api\/.+\.(js|ts)$/.test(file);
}

function routeNameFor(file) {
  const route = file
    .replace(/^app\//, "/").replace(/^pages\//, "/")
    .replace(/\/page\.(jsx|tsx|js|ts)$/, "").replace(/\/route\.(js|ts)$/, "")
    .replace(/\.(jsx|tsx|js|ts)$/, "").replace(/\/index$/, "/");
  return route || "/";
}

async function prismaFacts(repoPath, files) {
  const schemaFile = files.find((file) => file === "prisma/schema.prisma");
  if (!schemaFile) return [];
  const contents = await readFile(path.join(repoPath, schemaFile), "utf8");
  const models = [...contents.matchAll(/^model\s+([A-Za-z0-9_]+)\s+\{/gm)];
  return models.map((match) => ({
    kind: "db_model", name: match[1], evidence: [{ file: schemaFile }], layer: "heuristic"
  }));
}

async function envFacts(repoPath, files) {
  const facts = [];
  for (const file of files) {
    if (!isTextFile(file)) continue;
    const absolutePath = path.join(repoPath, file);
    const fileStat = await stat(absolutePath);
    if (fileStat.size > 250_000) continue;
    const contents = await readFile(absolutePath, "utf8");
    const envVars = [
      ...contents.matchAll(/process\.env\.([A-Z0-9_]+)/g),
      ...contents.matchAll(/import\.meta\.env\.([A-Z0-9_]+)/g)
    ];
    for (const match of envVars) {
      facts.push({ kind: "env_var", name: match[1], evidence: [{ file }], layer: "heuristic" });
    }
  }
  return dedupeFacts(facts);
}

async function maybeAddTextHints(repoPath, files, facts) {
  const hintPatterns = [
    { tag: "auth", pattern: /\b(auth|login|logout|signup|session|clerk|next-auth)\b/i },
    { tag: "payment", pattern: /\b(stripe|checkout|subscription|billing|webhook)\b/i },
    { tag: "email", pattern: /\b(resend|postmark|sendgrid|nodemailer|sendMail)\b/i },
    { tag: "notification", pattern: /notifications?|notify|bell|inbox/i },
    { tag: "permission", pattern: /\b(admin|role|permission|rbac)\b/i }
  ];
  for (const file of files) {
    if (!isTextFile(file)) continue;
    const absolutePath = path.join(repoPath, file);
    const fileStat = await stat(absolutePath);
    if (fileStat.size > 120_000) continue;
    const contents = await readFile(absolutePath, "utf8");
    for (const hint of hintPatterns) {
      if (hint.pattern.test(contents) || hint.pattern.test(file)) {
        facts.push({ kind: "code_hint", name: hint.tag, evidence: [{ file }], layer: "heuristic" });
      }
    }
  }
}

function packageTags(name) {
  const lower = name.toLowerCase();
  const tags = [];
  if (lower.includes("stripe")) tags.push("payment");
  if (lower.includes("clerk") || lower.includes("auth") || lower.includes("supabase")) tags.push("auth");
  if (["resend", "nodemailer", "postmark", "@sendgrid/mail"].includes(lower)) tags.push("email");
  if (lower.includes("prisma") || lower.includes("drizzle") || lower.includes("supabase")) tags.push("database");
  return tags;
}

function isTextFile(file) {
  return TEXT_FILE_EXTENSIONS.has(path.extname(file)) || path.basename(file).startsWith(".env");
}
```

```js
import path from "node:path";
import { readdir } from "node:fs/promises";
import { extract as extractNextjs } from "./extractors/nextjs.js";

const IGNORED_DIRS = new Set([
  ".git", ".next", ".varai", "build", "coverage", "dist", "node_modules",
  "__pycache__", ".venv", "venv", ".pytest_cache", ".mypy_cache"
]);

const TEXT_FILE_EXTENSIONS = new Set([
  ".cjs", ".css", ".env", ".js", ".jsx", ".json", ".md", ".mjs",
  ".prisma", ".py", ".sql", ".toml", ".ts", ".tsx", ".txt", ".yaml", ".yml"
]);

export async function scanRepo(repoPath, options = {}) {
  const include = options.include ?? [];
  const files = await walk(repoPath, include);
  const facts = await extractNextjs(repoPath, files);
  return {
    summary: { fileCount: files.length, factCount: facts.length },
    files,
    facts
  };
}

async function walk(root, include = []) {
  const files = [];
  async function visit(dir) {
    const entries = await readdir(dir, { withFileTypes: true });
    for (const entry of entries) {
      if (entry.isDirectory() && IGNORED_DIRS.has(entry.name)) continue;
      const absolutePath = path.join(dir, entry.name);
      const relativePath = path.relative(root, absolutePath);
      if (entry.isDirectory()) {
        if (include.length === 0 || include.some((prefix) => relativePath.startsWith(prefix) || prefix.startsWith(relativePath))) {
          await visit(absolutePath);
        }
      } else if (entry.isFile()) {
        if (include.length === 0 || include.some((prefix) => relativePath.startsWith(prefix))) {
          if (TEXT_FILE_EXTENSIONS.has(path.extname(entry.name)) || entry.name.startsWith(".env")) {
            files.push(relativePath);
          }
        }
      }
    }
  }
  await visit(root);
  return files.sort();
}
```

```js
import { readFile } from "node:fs/promises";
import { join } from "node:path";

export async function detectStacks(repoPath) {
  const stacks = new Set();
  const [packageJson, pyprojectToml, requirementsTxt, nextConfig] = await Promise.all([
    tryRead(join(repoPath, "package.json")),
    tryRead(join(repoPath, "pyproject.toml")),
    tryRead(join(repoPath, "requirements.txt")),
    tryRead(join(repoPath, "next.config.js")).then(v => v ?? tryRead(join(repoPath, "next.config.ts"))).then(v => v ?? tryRead(join(repoPath, "next.config.mjs")))
  ]);

  if (nextConfig !== null) stacks.add("nextjs"); // also added via package.json "next" dep below

  if (packageJson !== null) {
    try {
      const parsed = JSON.parse(packageJson);
      const allDeps = { ...(parsed.dependencies ?? {}), ...(parsed.devDependencies ?? {}) };
      if (nextConfig || "next" in allDeps) stacks.add("nextjs");
      if (!("next" in allDeps) && ("vite" in allDeps || "react" in allDeps)) stacks.add("react-vite");
    } catch { /* ignore malformed package.json */ }
  }

  if (pyprojectToml !== null) {
    stacks.add("python-common");
    if (/\bfastapi\b/i.test(pyprojectToml)) stacks.add("fastapi");
    if (/\bsqlalchemy\b/i.test(pyprojectToml)) stacks.add("sqlalchemy");
  }

  if (requirementsTxt !== null) {
    if (/^fastapi[=><!]/im.test(requirementsTxt) || /^fastapi$/im.test(requirementsTxt)) stacks.add("fastapi");
    if (/^sqlalchemy[=><!]/im.test(requirementsTxt) || /^sqlalchemy$/im.test(requirementsTxt)) stacks.add("sqlalchemy");
    if (pyprojectToml === null) stacks.add("python-common");
  }

  return stacks;
}

async function tryRead(filePath) {
  try {
    return await readFile(filePath, "utf8");
  } catch {
    return null;
  }
}
```

```js
import path from "node:path";
import { readFile, stat } from "node:fs/promises";
import { queryCaptures } from "../treesitter.js";

// Reads method + first string arg from an already-validated decorator node's text.
const DECORATOR_PARTS = /^@(?:app|router)\.(get|post|put|patch|delete|head|options)\s*\(\s*["']([^"']+)["']/i;

export async function extract(repoPath, files) {
  const facts = [];

    try {
      const fileStat = await stat(absolutePath);
      if (fileStat.size > 500_000) continue;
      content = await readFile(absolutePath, "utf8");
    } catch {
      continue;
    }

    const captures = await queryCaptures("python", content, "(decorator) @dec");
    for (const { node } of captures) {
      const parts = node.text.match(DECORATOR_PARTS);
      if (!parts) continue;
      const method = parts[1].toUpperCase();
      const routePath = parts[2];
      const line = node.startPosition.row + 1;
      const name = `${method} ${routePath}`;
      facts.push({ kind: "api_route", name, evidence: [{ file, line }], layer: "ast" });
      if (/webhook/i.test(routePath)) {
        facts.push({ kind: "webhook_route", name, evidence: [{ file, line }], layer: "ast" });
      }
    }
  }
  return facts;
```

- [ ] **Step 3: Create `src/scanners/extractors/sqlalchemy.js`**

Tree-sitter gives real `(class_definition)` nodes with a `name:` field and a `superclasses:` argument list; we keep classes whose superclasses mention `Base`. Migration files are detected by path (a legitimate convention), tagged `heuristic`.

```js
import path from "node:path";
import { readFile, stat } from "node:fs/promises";
import { queryCaptures } from "../treesitter.js";

const CLASS_QUERY = "(class_definition name: (identifier) @name superclasses: (argument_list) @supers)";

export async function extract(repoPath, files) {
  const facts = [];
  for (const file of files) {
    if (file.startsWith("alembic/versions/") || /\/alembic\/versions\//.test(file)) {
      if (path.extname(file) === ".py") {
        facts.push({ kind: "database_migration", name: path.basename(file), evidence: [{ file }], layer: "heuristic" });
      }
      continue;
    }
    if (path.extname(file) !== ".py") continue;
    const absolutePath = path.join(repoPath, file);
    let content;
    try {
      const fileStat = await stat(absolutePath);
      if (fileStat.size > 500_000) continue;
      content = await readFile(absolutePath, "utf8");
    } catch {
      continue;
    }

    // Captures come paired per match: a @name and a @supers for each class.
    const captures = await queryCaptures("python", content, CLASS_QUERY);
    const byMatch = pairCaptures(captures, ["name", "supers"]);
    for (const { name: nameNode, supers: supersNode } of byMatch) {
      if (!nameNode || !supersNode) continue;
      const name = nameNode.text;
      if (name === "Base") continue;
      if (!/\bBase\b/.test(supersNode.text)) continue;
      facts.push({ kind: "db_model", name, evidence: [{ file, line: nameNode.startPosition.row + 1 }], layer: "ast" });
    }
  }
  return facts;
}

// Group a flat capture list into per-match objects keyed by capture name.
// tree-sitter returns captures in document order; each new occurrence of the
// first key starts a new group.
function pairCaptures(captures, keys) {
  const [firstKey] = keys;
  const groups = [];
  let current = null;
  for (const { name, node } of captures) {
    if (name === firstKey) {
      current = {};
      groups.push(current);
    }
    if (current) current[name] = node;
  }
  return groups;
}
```

Tree-sitter confirms a real `create()` call and a real `zustand` import (not text in a comment), and finds `<Route>` JSX elements (both self-closing and opening forms). A store detected only by directory convention — with no parser-confirmed `create()` call — is tagged `heuristic`, not `ast`.

```js
import path from "node:path";
import { readFile, stat } from "node:fs/promises";
import { queryCaptures } from "../treesitter.js";

// .tsx needs the tsx grammar; .ts the typescript grammar; the rest are javascript (handles jsx).
const LANG_FOR_EXT = {
  ".js": "javascript", ".jsx": "javascript", ".mjs": "javascript", ".cjs": "javascript",
  ".ts": "typescript", ".tsx": "tsx"
};
const ROUTE_PATH = /path\s*=\s*["']([^"']+)["']/;

export async function extract(repoPath, files) {
  const facts = [];
  for (const file of files) {
    const lang = LANG_FOR_EXT[path.extname(file)];
    if (!lang) continue;
    const absolutePath = path.join(repoPath, file);
    let content;
    try {
      const fileStat = await stat(absolutePath);
      if (fileStat.size > 500_000) continue;
      content = await readFile(absolutePath, "utf8");
    } catch {
      continue;
    }

    const inStoreDir = file.includes("/store/") || file.includes("/stores/");

    let hasZustandImport = false;
    for (const { node } of await queryCaptures(lang, content, "(import_statement source: (string) @src)")) {
      if (node.text.includes("zustand")) { hasZustandImport = true; break; }
    }

    let hasCreateCall = false;
    for (const { node } of await queryCaptures(lang, content, "(call_expression function: (identifier) @fn)")) {
      if (node.text === "create") { hasCreateCall = true; break; }
    }

    if (inStoreDir || (hasZustandImport && hasCreateCall)) {
      const layer = hasCreateCall ? "ast" : "heuristic";
      facts.push({ kind: "state_store", name: path.basename(file, path.extname(file)), evidence: [{ file }], layer });
    }

    for (const { node } of await queryCaptures(lang, content, "[(jsx_self_closing_element) (jsx_opening_element)] @el")) {
      if (node.childForFieldName("name")?.text !== "Route") continue;
      const m = node.text.match(ROUTE_PATH);
      if (!m) continue;
      facts.push({ kind: "page", name: m[1], evidence: [{ file, line: node.startPosition.row + 1 }], layer: "ast" });
    }
  }
  return facts;
}
```

```js
const ENV_RE = /os\.(?:environ\[["']|environ\.get\s*\(\s*["']|getenv\s*\(\s*["'])([A-Z0-9_]+)["']/g;

export async function extract(repoPath, files) {

  for (const file of files) {

    } else if (ext === ".py") {
      facts.push(...await extractPythonEnvVars(repoPath, file));
    }
  }
  return dedupeFacts(facts);
}

async function extractPyproject(repoPath, file) {
  let content;
  try {
    content = await readFile(path.join(repoPath, file), "utf8");
  } catch {
    return [];
  }
  const facts = [];
  // Find the dependencies section
  const poetryDepSection = content.match(/\[tool\.poetry\.dependencies\]([\s\S]*?)(?=\[|$)/i);
  const projectDepSection = content.match(/\[project\][\s\S]*?dependencies\s*=\s*\[([\s\S]*?)\]/i);

  if (poetryDepSection) {
    for (const match of poetryDepSection[1].matchAll(POETRY_DEP_RE)) {
      const name = match[1].toLowerCase();
      if (name === "python") continue;
      facts.push({ kind: "package", name, evidence: [{ file }], layer: "heuristic" });
    }
  }
  if (projectDepSection) {
    for (const match of projectDepSection[1].matchAll(PROJECT_DEP_RE)) {
      facts.push({ kind: "package", name: match[1].toLowerCase(), evidence: [{ file }], layer: "heuristic" });
    }
  }
  return facts;
}

async function extractPythonEnvVars(repoPath, file) {

    for (const match of content.matchAll(ENV_RE)) {
      facts.push({ kind: "env_var", name: match[1], evidence: [{ file }], layer: "heuristic" });
    }
    return facts;
  } catch {
    return [];
  }
}
```

```js
import path from "node:path";
import { readdir } from "node:fs/promises";
import { detectStacks } from "./stack-detect.js";
import { extract as extractNextjs } from "./extractors/nextjs.js";
import { extract as extractFastapi } from "./extractors/fastapi.js";
import { extract as extractSqlalchemy } from "./extractors/sqlalchemy.js";
import { extract as extractReactVite } from "./extractors/react-vite.js";
import { extract as extractPythonCommon } from "./extractors/python-common.js";

const IGNORED_DIRS = new Set([
  ".git", ".next", ".varai", "build", "coverage", "dist", "node_modules",
  "__pycache__", ".venv", "venv", ".pytest_cache", ".mypy_cache"
]);

const TEXT_FILE_EXTENSIONS = new Set([
  ".cjs", ".css", ".env", ".js", ".jsx", ".json", ".md", ".mjs",
  ".prisma", ".py", ".sql", ".toml", ".ts", ".tsx", ".txt", ".yaml", ".yml"
]);

const EXTRACTOR_MAP = {
  "nextjs": extractNextjs,
  "fastapi": extractFastapi,
  "sqlalchemy": extractSqlalchemy,
  "react-vite": extractReactVite,
  "python-common": extractPythonCommon
};

export async function scanRepo(repoPath, options = {}) {
  const include = options.include ?? [];
  const files = await walk(repoPath, include);
  const stacks = await detectStacks(repoPath);

  const allFacts = [];
  // Fallback: if no stack detected at all, run nextjs (safe — patterns are Next.js-specific path conventions)
  if (stacks.size === 0) stacks.add("nextjs");

  for (const [stack, extractFn] of Object.entries(EXTRACTOR_MAP)) {
    if (stacks.has(stack)) {
      const facts = await extractFn(repoPath, files);
      allFacts.push(...facts);
    }
  }

  return {
    summary: { fileCount: files.length, factCount: allFacts.length },
    files,
    facts: allFacts
  };
}

async function walk(root, include = []) {
  const files = [];
  async function visit(dir) {
    const entries = await readdir(dir, { withFileTypes: true });
    for (const entry of entries) {
      if (entry.isDirectory() && IGNORED_DIRS.has(entry.name)) continue;
      const absolutePath = path.join(dir, entry.name);
      const relativePath = path.relative(root, absolutePath);
      if (entry.isDirectory()) {
        if (include.length === 0 || include.some((p) => relativePath.startsWith(p) || p.startsWith(relativePath))) {
          await visit(absolutePath);
        }
      } else if (entry.isFile()) {
        if (include.length === 0 || include.some((p) => relativePath.startsWith(p))) {
          if (TEXT_FILE_EXTENSIONS.has(path.extname(entry.name)) || entry.name.startsWith(".env")) {
            files.push(relativePath);
          }
        }
      }
    }
  }
  await visit(root);
  return files.sort();
}
```

```js
import path from "node:path";

export function renderInventory({ repoPath, scan }) {
  const name = path.basename(repoPath);
  const lines = [`# App Map — ${name}`, ""];

  const byKind = groupByKind(scan.facts);

  appendItemSection(lines, "API Routes", byKind.api_route ?? []);
  appendItemSection(lines, "Webhook Routes", byKind.webhook_route ?? []);
  appendItemSection(lines, "Pages", byKind.page ?? []);
  appendItemSection(lines, "Data Models", byKind.db_model ?? []);
  appendItemSection(lines, "Database Migrations", byKind.database_migration ?? []);
  appendItemSection(lines, "Frontend Stores", byKind.state_store ?? []);
  appendListSection(lines, "Packages", byKind.package ?? []);
  appendListSection(lines, "Env Vars", byKind.env_var ?? []);

  return `${lines.join("\n")}\n`;
}

function groupByKind(facts) {
  const groups = {};
  for (const fact of facts) {
    (groups[fact.kind] ??= []).push(fact);
  }
  return groups;
}

function appendItemSection(lines, title, facts) {
  if (facts.length === 0) return;
  lines.push(`## ${title} (${facts.length})`, "");
  for (const fact of facts) {
    const loc = evidenceRef(fact.evidence?.[0]);
    lines.push(`  ${fact.name.padEnd(40)} ${loc}`);
  }
  lines.push("");
}

function appendListSection(lines, title, facts) {
  if (facts.length === 0) return;
  lines.push(`## ${title}`, "");
  lines.push(`  ${facts.map((f) => f.name).join(", ")}`);
  lines.push("");
}

function evidenceRef(ev) {
  if (!ev) return "";
  return ev.line ? `${ev.file}:${ev.line}` : ev.file;
}
```

```js
import { runMap } from "../src/map.js";

const args = process.argv.slice(2);
const command = args[0];

function usage() {
  return `Varai — a lens for your codebase

Usage:
  varai map [<repo-path>] [--include <prefix>]...

Options:
  --include <prefix>   Only scan files under this path prefix (repeatable)

Examples:
  varai map
  varai map ../kalakar
  varai map ../kalakar --include services/backend --include services/frontend/src
  varai help
`;
}

function parseMapOptions(argv) {
  const options = { include: [] };
  for (let i = 0; i < argv.length; i++) {
    if (argv[i] === "--include" && argv[i + 1]) {
      options.include.push(argv[++i]);
    } else if (!argv[i].startsWith("--")) {
      options.repo = argv[i];
    } else {
      throw new Error(`Unknown option: ${argv[i]}\n\n${usage()}`);
    }
  }
  return options;
}

async function main() {
  if (!command || command === "help" || command === "--help" || command === "-h") {
    process.stdout.write(usage());
    return;
  }

  if (command !== "map") {
    process.stderr.write(`Unknown command: ${command}\n\n${usage()}`);
    process.exitCode = 1;
    return;
  }

  const options = parseMapOptions(args.slice(1));
  await runMap(options);
}

main().catch((error) => {
  process.stderr.write(`${error.message}\n`);
  process.exitCode = 1;
```

```markdown
# Varai Spec

Varai scans a repo and produces a structured inventory of what it contains. One command, no configuration required.

## What it does

`varai map <repo>` walks the repo, detects stacks (Next.js, FastAPI, SQLAlchemy, React/Vite, Python), runs per-stack extractors, and renders a markdown inventory grouped by category. Every line traces to a source file.

The Python and React extractors parse with **tree-sitter** (`web-tree-sitter` + WASM grammars), not regex over raw text — so a route in a comment, a multiline decorator, or a string literal never produces a false fact. The Next.js extractor uses reliable file-path conventions.

## Fact types

| Kind | Description |
|---|---|
| `api_route` | HTTP route (method + path) |
| `webhook_route` | Route whose path matches `/webhook/` |
| `page` | Frontend page or route |
| `db_model` | ORM model class |
| `database_migration` | Migration file |
| `state_store` | Frontend state store (Zustand) |
| `package` | Package dependency |
| `env_var` | Environment variable reference |

## Evidence layer

Every fact records how honestly it was found:

- `"ast"` — confirmed by a tree-sitter parse tree
- `"heuristic"` — read from a path convention, manifest, or text scan
- `"semantic"` — **reserved for the Phase 2 SCIP graph** (cross-file resolved edges); not emitted yet

Consumers (the renderer) never branch on the layer; it just keeps the report honest about confidence.

## Direction

`varai map` is altitude 1: a flat inventory of *what exists*. The destination is altitude 2 — the **structural graph** ("this route writes this model, which this page reads"), drawn with SCIP indexers (`scip-python`, `scip-typescript`, `scip-clang`) that resolve symbols across files. Tree-sitter locates nodes; SCIP resolves the edges between them. Naming a cluster as a capability is altitude 3 and is *not* recoverable from code alone — it requires supplied intent.

## What it is not

- Not a code reviewer
- Not a security scanner
- Not a test coverage tool
- Not an AI-assisted Q&A tool

Those are valid tools. This is not them.
```
